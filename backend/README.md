# Secret Santa Backend

Backend API для приложения Тайный Санта.

## Установка

```bash
npm install
```

## Разработка

```bash
npm run dev
```

Сервер запустится на порту 3001 (или на порту из переменной окружения PORT).

## Сборка

```bash
npm run build
```

## Запуск в продакшене

```bash
npm start
```

## Переменные окружения

Создайте файл `.env` на основе `.env.example`:

- `PORT` - порт сервера (по умолчанию 3001)
- `JWT_SECRET` - секретный ключ для JWT токенов (ОБЯЗАТЕЛЬНО измените в продакшене!)

## База данных

Приложение использует простой JSON файл (`data.json`) для хранения данных. Файл создаётся автоматически при первом запуске.

## Структура данных

```typescript
{
  users: User[];      // Пользователи
  rooms: Room[];      // Комнаты
  wishlists: Wishlist[]; // Списки желаний
}
```
